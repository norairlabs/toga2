ROTOR BRAKE ACTIVE,Whether the rotor brake is active (1, TRUE) or not (0, FALSE).
ROTOR CLUTCH ACTIVE,Whether the rotor clutch is active (1, TRUE) or not (0, FALSE).
ROTOR CLUTCH SWITCH POS,The rotor clutch switch position, either on (1 TRUE) or off (0, FALSE).
ROTOR GOV ACTIVE:1,Whether the rotor governor is active (1, TRUE) or not (0, FALSE) on #1 engine.
ROTOR GOV ACTIVE:2,Whether the rotor governor is active (1, TRUE) or not (0, FALSE) on #2 engine.
IS ATTACHED TO SLING,Set to true if this object is attached to a sling.
SLING CABLE BROKEN:1,THis will be True (1) if the #1 cable is broken, or False (0) otherwise.
SLING CABLE BROKEN:2,THis will be True (1) if the #2 cable is broken, or False (0) otherwise.
SLING HOIST SWITCH:1,This will be True (1) if the #1 hoist is enabled or False (0) otherwise.
SLING HOIST SWITCH:2,This will be True (1) if the #2 hoist is enabled or False (0) otherwise.
SLING HOOK IN PICKUP MODE,This will be True (1) if the hook is in pickup mode or False (0) otherwise. When True, the hook will be capable of picking up another object.
